Game Design - Glorbo's Basement
Multiplayer Dungeon Crawler game with an Isometric view.
Project Details / Background
Game Design project where I worked in a team of 4 to make a networked multiplayer dungeon crawler game dubbed Glorbo's Basement. The game is fantasy theme and features basic combat and a class system as well as items and powerups that players can pick up as they progress through several floors.
I worked on the tilemap and created the conversion of cartesian coordinates to our own isometric coordinate system so all of the tiles, players, enemies, and projectiles are rendered in the correct order and position. Because of this I handled all of the map collisions and pathfinding for the enemies with A Star algorithm.
